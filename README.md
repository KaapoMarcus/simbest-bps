# simbest-bps
##集成普元BPS工作流引擎


#Maven 仓库添加
```
<dependency>
	<groupId>com.simbest</groupId>
	<artifactId>simbest-bps</artifactId>
	<version>0.6</version>
</dependency>
		
		
<repositories>
	<repository>
		<id>simbest-bps-mvn-repo</id>
		<url>https://raw.github.com/simbest/simbest-bps/mvn-repo/</url>
		<snapshots>
			<enabled>true</enabled>
			<updatePolicy>always</updatePolicy>
		</snapshots>
	</repository>
</repositories>	
```
